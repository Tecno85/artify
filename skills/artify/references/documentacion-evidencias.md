# Documentación Y Evidencias

Leer esta referencia al trabajar en `docs/`, `README.md`, `CONTEXT.md`, manuales, diagramas, capturas o entregas académicas.

## Coherencia Documental

- Usar `docs/proyecto/` para material funcional o académico y `docs/tecnica/` para arquitectura, base de datos, despliegue, pruebas, analytics y estándares.
- Mantener implementado y futuro separados; no prometer demo pública, atajos, formatos u otras funciones que no existan.
- Actualizar documentación con el cambio de comportamiento correspondiente, sin reescribir documentos que continúen vigentes.
- Usar `README.md` como índice documental principal y `CONTEXT.md` como estado operativo; evitar duplicar detalles extensos entre ambos.
- Verificar cifras variables, versiones, nombres de archivos, rutas y enlaces contra el repositorio antes de publicarlos.

## Evidencias Visuales

- Crear o modificar archivos en `docs/**/evidencias/` solo cuando el usuario lo pida o la entrega lo requiera.
- Localizar manuales y capturas desactualizados cuando cambie una interfaz documentada.
- Distinguir explícitamente una captura real, una representación visual sanitizada y un diagrama explicativo; no presentar una reconstrucción como captura real.
- Usar datos ficticios o anonimizados. Excluir credenciales, tokens, cadenas de conexión, correos reales y datos personales.
- Mantener coherentes imagen, pie, texto alternativo, instrucciones y estado real mostrado.
- Verificar visualmente documentos y evidencias cuando el formato o la composición importen.

## Respuesta Al Instructor

Cuando una evidencia quede creada como artefacto Markdown, preparar un texto breve con datos reales:

```text
Buen día, Instructor Jose.

Hago envío de la evidencia [código y nombre de la evidencia].

El informe lo desarrollé como artefacto Markdown dentro del repositorio del proyecto Artify y se encuentra en la siguiente ubicación:

[ruta del archivo dentro del repositorio]

El link para visualizarlo es: [[código de la evidencia]](https://github.com/Tecno85/artify/blob/main/[ruta-del-archivo])

Quedo atento a sus comentarios.

Muchas gracias.

Bendiciones.
```

Sustituir el código, nombre, ruta y enlace; no entregar marcadores pendientes.
